Run one command or a sequential command pipeline on a live micrOS device. First use `search_devices` to confirm the target and relevant signature, then pass an exact UID, device name, or IP. Commands can change real device state. Configuration reads (`conf`, `conf key`) are allowed, but configuration writes are rejected before connection; other commands are not made safe automatically. A string may separate pipeline steps with `<a>`, or `command` may be an array.
