Use micrOSMCP to find and operate micrOS devices on the local network. Start with search_devices to identify a target and inspect cached command signatures; use list_devices only for a compact inventory. If devices or capabilities are missing or stale, run discover_devices or discover_commands, then search again. Before run_command, confirm the exact UID, name, or IP and choose the least disruptive command that satisfies the request: commands run on real devices and may change their state. Use set_device_note to preserve helpful device context.
