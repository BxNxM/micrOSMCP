Find and operate micrOS devices on the local network. Use `list_devices` for a quick inventory and `search_devices` to find a specific device or capability. If a device is missing, use `discover_devices`; if its available commands are unknown or outdated, use `discover_commands`. Before calling `run_command`, confirm the target and command signature with `search_devices`. Commands run on real devices and may change their state, so choose the least disruptive command that satisfies the request. Use `set_device_note` to save helpful context such as a device's location, attached hardware, or purpose.
